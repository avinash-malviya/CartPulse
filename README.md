# CartPulse: Apparel Intelligence

**CartPulse: Apparel Intelligence** is a premium, enterprise-grade operational intelligence platform designed for high-growth D2C clothing brands in India. It provides real-time observability into the customer journey, operational health, and revenue leak detection.

## 🚀 Key Features

- **Real-time KPI Monitoring**: Track GMV at Risk, UPI Success Rates, and Cart Abandonment in real-time.
- **Purchase Funnel Observability**: Identify friction points from Homepage to Payment Success.
- **API & Service Health**: Monitor critical services like Razorpay, Delhivery, and Inventory Sync.
- **Regional Intelligence**: Localized insights for the Indian market (Maharashtra, South India, etc.).
- **Incident Troubleshooting**: Deep-dive into failed sessions to identify root causes like UPI timeouts or Pincode serviceability issues.
- **Demo Simulation Center**: Trigger various operational scenarios (Payment Outage, Flash Sale, Inventory Sync Issues) to see the platform in action.

## 🛠️ Tech Stack

- **React 19** & **TypeScript**
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Recharts** (Data Visualization)
- **shadcn/ui** (UI Components)
- **Vite** (Build Tool)

## 📦 Deployment on Netlify

This project is optimized for deployment on **Netlify**.

1. **Export to GitHub**: Use the AI Studio export feature to push this code to a GitHub repository.
2. **Connect to Netlify**:
   - Log in to [Netlify](https://netlify.com).
   - Click **"Add new site"** -> **"Import an existing project"**.
   - Select your GitHub repository.
3. **Configure Settings**:
   - Build Command: `npm run build`
   - Publish Directory: `dist`
   - (The included `netlify.toml` handles these automatically).
4. **Deploy**: Click **"Deploy site"**.
